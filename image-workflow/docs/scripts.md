# Scripts / 脚本说明

Scripts are not required for `image-workflow`.

脚本不是必须的。

## Why Add Scripts? / 为什么以后可以加脚本？

Scripts are useful when you want repeated tasks to be more stable and token-efficient:

- Generate contact sheets from a folder.
- Check whether the same photo is used twice.
- Create a source-selection list.
- Validate output image dimensions.
- Build a `style-index.md` draft from filenames.
- Clean temporary files.

脚本适合处理重复、机械、容易出错的任务，比如：

- 自动生成素材联系表。
- 检查同一张照片是否重复使用。
- 生成素材选择清单。
- 检查输出图片尺寸。
- 根据文件名草拟 `style-index.md`。
- 清理临时文件。

## Current Choice / 当前选择

The first open-source version keeps the project simple: it provides the skill and documentation, but no scripts.

第一版先保持轻量：只提供 skill 和文档，不强制脚本化。
