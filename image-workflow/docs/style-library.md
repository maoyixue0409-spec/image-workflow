# Style Library Guide

`image-workflow` works best when each user builds a small local style library.

## Folder Roles

- `styles/`: complete layout references, such as clean multi-photo collages, scrapbook pages, food layouts, and handwritten-title covers.
- `elements/`: individual visual elements, such as doodles, stars, ribbons, bows, frames, hand-drawn labels, and sticker sheets.
- `backgrounds/`: reusable clean backgrounds, such as scenery photos, paper textures, fabric textures, walls, sky, or soft color fields.
- `style-index.md`: short notes that help Codex understand when each style reference is useful.

## Style Index Template

```markdown
# Style Index

- `clean-multi-photo-layout.jpg`: full-photo collage, light handwritten title, low doodle density, good for travel diaries.
- `food-cutout-layout.jpg`: food stickers with white edge, cafe background, good for food trips.
- `retro-scrapbook.jpg`: warmer tones, film texture, tape and labels, good for nostalgic diary posts.
```

## Asset Rules

- Use your own images or assets you have permission to use.
- Do not publish private portraits, private travel photos, or unlicensed inspiration images in an open repository.
- If an image is inspiration only, write notes in `style-index.md` and keep the image private.
