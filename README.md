image-workflow 是一个 Codex skill，用于根据本地照片、风格关键词、标题文案和用户自己的风格素材库，生成社交媒体拼贴图片。
它适合想把旅行、美食、咖啡馆、生活方式、日记照片做成小红书 / RedNote 图文封面或拼贴图的人。

这是什么
image-workflow 不是一套固定模板，而是一套可复用的 Codex 工作流说明。
安装后，你可以在 Codex 中要求使用这个 skill。它会引导 Codex 读取你的照片、检索你的素材库参考图、先给出简短方案，等你确认后再生成最终图片。

有什么用
你可以用 image-workflow 生成：
·小红书 / RedNote 旅行封面
·生活日记拼贴图
·美食旅行和咖啡馆图文
·手帐感 / 拼贴感多图排版
·清爽手写标题图片
·根据 moodboard 延展出的社交媒体视觉

怎么安装
把整个文件夹复制到 Codex skills 目录：
mkdir -p ~/.codex/skills
cp -R image-workflow ~/.codex/skills/image-workflow
安装后重启 Codex。
然后你可以这样触发：
使用 image-workflow。

怎么使用
1、准备你的原始照片。
2、准备一个风格素材库，里面可以放排版参考、涂鸦、背景图或原创参考板。
3、在 Codex 里要求使用 image-workflow。
4、提供风格关键词、标题、照片文件夹、输出数量和特殊要求。
5、让 Codex 先检查文件并给出简短方案。
6、你确认或调整方案。
7、再生成最终图片。

复制即用启动词
使用 image-workflow。

我想用本地照片生成社交媒体拼贴图片。
请你帮我：
1. 根据我的风格关键词，从我的风格素材库里找到匹配的参考图。
2. 先查看参考图，再设计。
3. 背景默认保持干净，不要明显模糊，除非我明确要求模糊。
4. 每张成品图里，同一张原始照片只能出现一次。
5. 合适时加入自然的食物抠图、手写感文字和不生硬的涂鸦。
6. 先给我简短方案，等我确认后再生成最终图片。
