# image-workflow
根据素材库风格关键词、旅行/生活照片、标题文案，生成小红书图文封面/拼贴图。
image-workflow 是一个 Codex Skill，用于根据本地照片、风格关键词和用户自己的素材库，生成适合小红书 / RedNote / 社交媒体发布的拼贴图片。

它适合旅行图文封面、生活日记拼贴、美食探店图片、咖啡馆图文、手帐风拼贴、Moodboard 风格图和个人照片整理。

使用时，用户可以提供风格关键词、标题、小字、原始照片文件夹和输出数量。Skill 会先从用户的风格素材库中查找匹配的参考图，查看参考图后给出简短生成方案，等用户确认后再生成最终图片。

image-workflow 的核心规则包括：同一张照片在一张成品图里只出现一次；背景图不会再被重复放进拼贴；背景默认不做明显模糊；食物抠图默认保留白边但不全部做成圆形；文字尽量保持清爽、手写感和自然排版；涂鸦和贴纸要避免生硬和过度 AI 感。

本项目不包含私人照片、Pinterest 参考图或未授权素材。使用者需要准备自己的照片和风格素材库，也可以用 Zine Studio 等工具先生成原创风格参考板，再配合 image-workflow 使用。

安装方式：将 image-workflow 文件夹复制到 ~/.codex/skills/image-workflow/，然后重启 Codex。在新对话中输入“使用 image-workflow”，并提供风格关键词、照片文件夹和标题即可开始。
